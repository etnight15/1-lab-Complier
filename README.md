# Разработка пользовательского интерфейса (GUI) для языкового процессора

## Цель работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

## Автор
Студент: Марков Д.Д.
Год: 2026

## Описание проекта
Приложение представляет собой текстовый редактор с графическим интерфейсом, реализующий базовые операции работы с текстовыми файлами. Включает главное меню, панель инструментов, область редактирования и область вывода результатов. Приложение разработано с учетом дальнейшего расширения функционала для создания языкового процессора.

## Используемые технологии
- **Язык программирования:** Python 3.14
- **GUI фреймворк:** PyQt6
- **Среда разработки:** VS Code
- **Дополнительные библиотеки:** 
  - PyQt6 - для создания графического интерфейса
  - sys, os - системные операции

## Инструкция по сборке и запуску

### Установка зависимостей
```bash
# Создание виртуального окружения (рекомендуется)
python -m venv venv

# Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
# Запуск
python main.py

```
